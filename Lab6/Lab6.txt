1. (Файл Lab6_1) Работа с JSON и XML:
    - JSON-пример демонстрирует чтение и обработку данных в формате JSON:
        - Преобразует JSON-строку в словарь Python
        - Извлекает и выводит информацию о книге
        - Проверяет наличие темы "data science"

    - XML-пример показывает парсинг XML-данных с помощью BeautifulSoup:
        - Извлекает информацию о книге из XML-структуры
        - Выводит заголовок, автора и список тем

2. (Файл Lab6_2) Анализ репозиториев GitHub:
    - Получает список репозиториев указанного пользователя через GitHub API
    - Анализирует даты создания репозиториев: показывает распределение по месяцам и дням недели
    - Выводит языки программирования из последних 5 репозиториев
    - Пример использования: анализ аккаунта "remeliora" (можно заменить на любого пользователя)

3. (Файл Lab6_3) Анализ новостей через NewsAPI:
    - Получает новости через NewsAPI с возможностью фильтрации:
        - По ключевым словам (query)
        - По категориям (например, "science")
        - По странам (например, "gb" для Великобритании)
    - Анализирует и выводит:
        - Топ источников новостей
        - Самых активных авторов
        - Даты публикаций
        - Примеры последних новостей с ссылками
    - Примеры запросов: научные новости, статьи об ИИ, технологические новости из UK
