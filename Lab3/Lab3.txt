Этот код создает серию графиков для визуализации данных с помощью библиотеки `matplotlib` и сохраняет их в папку `im`.
В этом коде:
1. Каждый график сохраняется в отдельный файл PNG вместо показа на экране
2. После сохранения график очищается (`plt.gca().clear()`) для следующего построения
3. Код демонстрирует разные типы визуализации: линейные, столбчатые, точечные
4. Подчеркивает важные аспекты визуализации: выбор масштаба, подписи, легенды

1. Линейный график номинального ВВП
- Строит график изменения ВВП (в млрд $) по годам с 1950 по 2010
- Использует зеленые точки с соединяющими линиями
- Сохраняет в файл `1_Номинальный_ВВП.png`

2. Столбчатая диаграмма фильмов
- Показывает количество наград Оскар для 5 фильмов
- Подписывает оси и названия фильмов
- Сохраняет в `2_Фильмы.png`

3. Гистограмма распределения оценок
- Группирует оценки студентов по децилям (0-10, 10-20, ..., 90-100)
- Визуализирует количество студентов в каждом диапазоне
- Сохраняет в `3_Оценки.png`

4-5. Демонстрация важности выбора масштаба осей
- Показывает два варианта визуализации одних и тех же данных (увеличение упоминаний науки о данных с 500 до 505)
- Первый график (`4_...`) использует обрезанную ось Y, создавая ложное впечатление резкого роста
- Второй (`5_...`) показывает те же данные в правильном масштабе
- Это демонстрация того, как можно манипулировать восприятием данных

6. Линейный график компромисса в моделировании
- Показывает три кривые: дисперсию, квадрат смещения и суммарную ошибку
- Демонстрирует концепцию компромисса между смещением и дисперсией в машинном обучении
- Сохраняет в `6_Линейная_диаграмма.png`

7. Точечная диаграмма (scatter plot) активности пользователей
- Отображает зависимость времени на сайте от количества друзей
- Каждая точка подписана буквой
- Сохраняет в `7_Диаграмма_рассеяния.png`

8-9. Сравнение оценок с разными осями
- Два графика одинаковых данных (оценки за два теста)
- Первый (`8_...`) с разными масштабами осей искажает восприятие
- Второй (`9_...`) с одинаковым масштабом (plt.axis("equal")) дает объективную картину
- Показывает важность правильного масштабирования осей
